## Conventions

- Every new script MUST be added to this manifest
- Tools are organized by workflow subdirectory (e.g., `tools/dx/`, `tools/build/`)
- Each tool does ONE thing — deterministic, testable, documented
- Tools never make decisions — they execute

*Add new tools here as they are created. One row per script.*
